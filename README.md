# 🦞 newsClaw

简单的 Flask 时间显示应用

## 功能

- 显示当前时间（上海时区）
- 自动每秒刷新
- 美观的渐变背景

## 安装

```bash
# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

## 运行

```bash
python app.py
```

访问 http://localhost:5000 查看页面

## 技术栈

- Python 3
- Flask
- pytz
